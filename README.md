# Image Encoder 🖼️

Utilitário desenvolvido em C para codificação e processamento de imagens.

## 📌 Objetivo

Explorar manipulação de arquivos binários, ponteiros e alocação dinâmica de memória.

## 🛠 Tecnologias Utilizadas

- Linguagem C
- Manipulação de arquivos
- Ponteiros
- Alocação dinâmica (malloc/free)

## ⚙️ Funcionalidades

- Leitura de arquivos de imagem
- Processamento de dados binários
- Escrita de arquivos transformados
- Gerenciamento manual de memória

## 🧠 Conceitos Aplicados

- Programação Estruturada
- Manipulação de Buffers
- Controle de Memória
- Lógica de Algoritmos

## 🚀 Como Executar

1. Clone o repositório:
   git clone https://github.com/Invencible-Maximilian/Image_encoder

2. Compile via terminal:
   gcc main.c -o encoder

3. Execute:
   ./encoder

## 📚 Aprendizados

- Importância do gerenciamento manual de memória
- Tratamento de erros em manipulação de arquivos
- Eficiência no processamento de dados binários
